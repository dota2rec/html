Item categorization

  • Consumable
  • Basic
    • Cost < 1000
  • Mid
    • 1001 < Cost < 3500
  • Advanced
    • 3501 < Cost
  • Other types
    • Crafted
      • We use manual dictionary to remove consumed item.
    • Upgradable
      • An item may have multiple levels, but have the same ID.

Algorithm

  • Hero-Item Matrix
  • Baseline: Win*1.2, Lose*0.8

    Perseverance’s item hierarchy: Its components are Ring of Health and Void Stone, and it is used to build Refresher Orb, Linken’s Sphere, Lotus Orb and Battle Fury.

  • Items that benefit yourself
  • Items that benefit other teammates
